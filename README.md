# BrebbleSnapGame
Snap Game for _Nology using Java.

## Objective
Making a simple card game of snap that can be played with one or two people.

## How it Works
When the game starts, it makes a deck of cards and shuffles it before revealing the first card of the deck. Then with each "turn" played, it reveals the next the card and checks to see whether the suit matches the previous card draw.
In a single player game, once this happens the game ends immediately.
In a two player game, the turn passed between each player. Then when a snap occurs, the turn player has 2 seconds to input "Snap" so that they can win the game. If they misspell it or dont do it quick enough, they lose and the other player wins instead.
